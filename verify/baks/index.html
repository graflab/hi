How This Verification Works

To verify your document, set the original document and its corresponding .vgti file. The system will check the file's authenticity using the Violet ID stored on-chain. If the file is verified successfully, you will be provided a link to view the transaction on the Cardano blockchain explorer of your choice (cexplorer.io or cardanoscan.io).
